02/08/2023 : L'IA dans les jeux vidéos

On definit souvent l'intelligence artificielle comme le processus realisant une action par une machine (ordinateur) afin de resoudre un probleme posé grace a l'utilisation d'un algorithme. Ce resultat se doit de se conformer a une action qu'une personne humaine realiserait pour le meme probleme.
Mais le juge, c'est la personne qui observe
Dans le domaine du jeu video, l'intelligence artificielle est utilisée afin de creer un monde.
