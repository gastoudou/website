Technical Dive : CDPR - CP77

Après un projet VRally4 qui a été plus ou moins réussi, je suis passé chez CD Projekt Red pour travailler sur l'IA des véhicules. Cela concernait les voitures, les motos et les camions.
Toutes les fois que le joueur est transporté par un PNJ (personnage non jouable), l'IA prend le contrôle.
Lorsque j'ai rejoint le projet, rien n'était prêt pour les véhicules dans le jeu. La démo présentée à l'E3 utilisait des splines et des animations pré-calculées pour montrer un semblant de trafic routier.
Le seul système présent était le système de Behaviours Tree pour les piétons, que nous voulions également utiliser pour les véhicules.
